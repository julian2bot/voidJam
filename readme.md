# Void Jam

Game Jam sur 2jours.... 

Dans ce jeu, vous êtes en voiture tard le soir et vous devez tenir le plus longtemps possible en restant éveillé tout en esquivant les obstacles et les murs.  

Récoltez du café et laissez la musique allumée pour vous fatiguer moins vite !  

Ce jeu a été réalisé dans une courte période et sans moteur. (Nous avons utilisé la SDL2 en C, avec la méthode de raycasting)  
  

## thème

> Accident de voiture
